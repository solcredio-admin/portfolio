Projects

Home Network Infrastructure - TP-Link Omada

Built a managed home network using a TP-Link ER605 router, an Omada 8-port switch, and an EAP110 outdoor access point - extending Wi-Fi coverage across the entire block. The network controller runs self-hosted on a local server, managing all access points from adoption through to configuration.

Handles static IP assignments, VLAN-style device segmentation, and connects all house devices including smart TVs and desktop workstations. All cabling is custom - hand-crimped RJ45 patch cables for the patch panel and outdoor-rated Cat6 runs for longer distances.

Proxmox Homelab - Virtualization & Self-Hosted Services

Running a Proxmox hypervisor host managing multiple VMs and LXC containers. Lab includes several Linux VMs for distro testing, a Windows Server instance, and an Active Directory domain controller with domain-joined VMs for hands-on practice with AD policies and user onboarding workflows.

A dedicated VM runs Docker hosting multiple self-hosted applications, including IT documentation tools and MSP-simulated tools such as a self-hosted remote desktop solution and a ticketing system - replicating a real MSP environment for practice and skill development.

NAS Build - TrueNAS Scale & Media Server

Repurposed a desktop into a NAS with four 1TB drives, testing multiple operating systems including TrueNAS Core, OpenMediaVault, and CasaOS before settling on TrueNAS Scale for its stability and app ecosystem.
